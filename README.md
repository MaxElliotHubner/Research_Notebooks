# Research Notebooks

This repository contains selected Python and Mathematica/Wolfram notebooks from my research projects.

## Projects

### New 4D N=2 SCFTs via Freezing
- Language: Python
- Tools: Jupyter, NumPy, Pandas, Sympy
- Description: Generalization of frozen ADE singularities to frozen CY3 singularities, the latter give new IIA string backgrounds engineering 4D N=2 SCFTs. Work in progress.

### 1-form Anomaly and Spectral Invariants
- Language: Python
- Tools: Jupyter, NumPy, Pandas, Sympy
- Description: We compute anomalies of generalized symmetries using extra-dimensional eta-invariants and give cross-checks via intersection theory for toric models. Preprint: https://arxiv.org/abs/2512.17906.

### Non-Supersymmetric Orbifolds
- Language: Python
- Tools: Jupyter, NumPy, Pandas, Sympy
- Description: Non-supersymmetric orbifolds in type II string theory are analyzed using brane probes. We compute defect groups from screening computations and establish that these can also be computed more efficiently directly from the geometric background using Chen-Ruan orbifold cohomology. Chen-Ruan orbifold cohomology geometrizes twisted sectors and captures otherwise inaccesible contributions. Publications: https://inspirehep.net/literature/2911705, https://inspirehep.net/literature/2781644

### Root Monodromy for Irregular Punctures
- Language: Python
- Tools: Jupyter, NumPy, Pandas, Sympy
- Description: Irregular punctures in class S can support their own set of line defect operators. These are encoded in Monodromy data of the puncture. We give a cute numeric method of computing the induced monodromy matrices on the relvant root lattices. Publication: https://inspirehep.net/literature/1999694

### Non-Abelian Flavor Symmetry Enhancement for 5D SCFTs
- Language: Python
- Tools: Jupyter, NumPy, Pandas, Sympy
- Description: Non-abelian flavor symmetries of a 5D SCFT may enhance at the origin of its Coulomb branch. We compute this enhancement for E-type conformal matter. Publication: https://inspirehep.net/literature/1798960
